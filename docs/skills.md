# 技能系统

## 开局三选二

游戏开始后先进入 `skill_selection` 阶段。每名玩家从 12 个技能中随机获得 3 个候选技能，只能看到自己的候选技能。玩家选择其中 2 个后完成准备；所有玩家都选完后，牌局进入 `playing` 阶段。

未完成技能选择前，不能摸牌、出牌、吃、碰、杠、胡或使用主动技能。

## 四巡一次

“四巡一次”简化为：该玩家每完成 4 次自己的出牌回合，才能再次使用。系统用 `turn_counts[player_id]` 记录完成回合数，并在技能使用时记录上次使用时的回合数。

适用技能：

- `astrology`
- `peek_neighbor`
- `change_suit`

## 反射范围

`mirror_reflection` 只反射目标型主动技能：

- `peek_neighbor`
- `swap_with_neighbor`
- `steal_concealed_gang`

它不反射非目标技能、被动技能，也不会被另一个镜像反射再次反射。

## 隐藏信息

观星、偷窥、偷天换日获得的牌、自行印牌结果、偷暗杠偷到的牌，都只进入使用者自己的 private view。其他玩家只能看到公开 action log 中的技能使用摘要。

## 当前简化点

- 四巡一次按玩家自己的完成回合数计算。
- 镜像反射只反射目标型主动技能。
- 偷暗杠不会重新结算已经发生过的杠分；同一个暗杠仍然只能被偷一次，但技能本身不限次数。
- 自行印牌失败仍消耗技能，并改为正常摸牌。
- 换色目标牌必须来自牌墙。
- 偷摸开杠使用手中三张相同牌，再指定一张任意牌组成暗杠；指定牌不从牌墙扣除，这是技能特殊效果。
